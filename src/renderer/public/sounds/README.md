# 効果音(SE)の配置

録音の開始・終了時に鳴らす効果音をここに置きます。

| ファイル | タイミング |
| --- | --- |
| `start.mp3` | 録音開始時 |
| `stop.mp3` | 録音終了時（正常完了。ESC キャンセル・エラー時は鳴りません） |

- 形式: `mp3` / `wav` / `ogg`（参照コードのファイル名・拡張子は `start.mp3` / `stop.mp3` 固定。変更する場合は `src/renderer/src/App.tsx` の `playSE()` 内のパスも合わせる）
- 長さの目安: 1 秒以内の短い音
- 未配置の場合は無音（再生失敗は握りつぶす）
- ON/OFF は設定画面の「録音の開始・終了時に効果音を鳴らす」で切り替え
